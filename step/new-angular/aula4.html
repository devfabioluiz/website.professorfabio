17. O que é navegação no Angular?

Navegação é o processo de mudar de uma tela para outra dentro da aplicação.

Em aplicações Angular, isso normalmente é feito com rotas.

18. O que são rotas?

Rotas definem qual componente será exibido quando um determinado caminho da URL for acessado.

Exemplo:

  • /home → página inicial
  • /produtos → página de produtos
  • /detalhe/1 → detalhe de um produto

19. Exemplo de rotas


import { Routes } from '@angular/router';
import { HomeComponent } from './home/home.component';
import { ProdutosComponent } from './produtos/produtos.component';

export const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'produtos', component: ProdutosComponent }
];

20. Navegação pelo template

Podemos navegar entre páginas usando a diretiva routerLink. 


Ir para produtos

21. Navegação pelo TypeScript

Também podemos navegar por código usando o serviço Router. 


import { Router } from '@angular/router';

constructor(private router: Router) {}

irParaProdutos() {
  this.router.navigate(['/produtos']);
}

22. Transferência de dados entre rotas

Existem várias formas de transferir dados de uma página para outra no Angular.

  • Parâmetros de rota
  • Query params
  • State de navegação
  • Uso de serviços compartilhados

23. Exemplo com parâmetro de rota


{ path: 'produto/:id', component: ProdutoDetalheComponent }
          

Ver produto 10

Nesse caso, o número 10 será enviado na URL como identificador do produto.

24. Lendo o parâmetro da rota


import { ActivatedRoute } from '@angular/router';

constructor(private route: ActivatedRoute) {}

ngOnInit() {
  const id = this.route.snapshot.paramMap.get('id');
  console.log(id);
}

Assim conseguimos recuperar o valor enviado na rota.
