Oracle e Postgre SQL

Módulo 3 - Semi-Integral

Versão 1.0.0

Duração do Curso – 24 horas

Objetivo do Curso

Ensinar os alunos a projetar bancos de dados usando Oracle e o PostgreSQL.

Ao Final do Curso, o Aluno Poderá

  • Descrever a arquitetura Oracle DBMS
  • Criar bancos de dados no Oracle DBMS
  • Programar em PL/SQL
  • Fazer backup de bancos de dados Oracle DBMS
  • Executar recuperação de banco de dados para Oracle DBMS
  • Instalar e configurar o PostgreSQL
  • Programar em PL/pgSQL
  • Usar construções avançadas de PL/pgSQL
  • Criar gatilhos, procedimentos armazenados, funções definidas pelo usuário

Avaliação do Curso

No final deste curso, o aluno precisa apresentar um Projeto Final. Para a admissão no Projeto, todos os trabalhos de casa e tarefas práticas devem estar carregados no sistema.
