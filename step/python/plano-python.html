Python Back-End

Versão 2.5.1

Duração do Curso – 94 horas

Objetivo do curso

Capacitar o participante em programação web utilizando Python.

Ao final do curso, o aluno será capaz de:

  • Compreender em profundidade o uso de tuplas e dicionários
  • Criar geradores
  • Aplicar funções lambda
  • Entender closures
  • Programar decoradores
  • Compreender os fundamentos de POO em Python
  • Desenvolver aplicações web com Python
  • Processar dados de formulários
  • Interagir com fontes de dados
  • Compreender MVC e MVT
  • Desenvolver aplicações com Django
  • Utilizar Django ORM
  • Trabalhar com roteamento
  • Criar APIs REST com Django REST Framework

Critérios de avaliação

Durante o curso, cada estudante desenvolverá um projeto individual. Ao final, deverá apresentar e defender o projeto e realizar um exame teórico.

📌 Para estar apto ao exame final, o estudante deverá entregar todas as atividades práticas.

Antes do início do curso, é obrigatório acesso ao curso:
Cisco NetAcad - Python Essentials 2

🎓 Preparação para certificação: PCAP – Certified Associate in Python Programming
