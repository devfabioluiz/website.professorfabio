1. Tuplas

Uma tupla é uma coleção de elementos ordenados e imutáveis em Python. Ela funciona de forma parecida com uma lista, porém seus valores não podem ser alterados após a criação.

Criando uma Tupla


coordenadas = (10, 20)

print(coordenadas)

Depois de criada, uma tupla não pode ter seus elementos alterados. 


numeros = (1, 2, 3)

numeros[0] = 10

O código acima gera um erro, pois tuplas são imutáveis.

Características das Tuplas

  • São imutáveis
  • Preservam a ordem dos elementos
  • Podem armazenar diferentes tipos de dados
  • São mais eficientes em memória que listas

Exemplo Prático

Um uso comum é armazenar coordenadas geográficas.


localizacao = (-22.9068, -43.1729)

latitude = localizacao[0]
longitude = localizacao[1]

print(latitude, longitude)

2. Dicionários

Um dicionário é uma estrutura de dados baseada em pares de chave e valor. 


pessoa = {
  "nome": "Fabio",
  "idade": 30,
  "cidade": "Rio"
}

print(pessoa["nome"])

Mapas Associativos

Dicionários também são chamados de mapas associativos, pois cada chave está associada a um valor.

Chave Valor
nome Fabio
idade 30
cidade Rio

Tabelas Hash

Internamente os dicionários utilizam uma estrutura chamada Hash Table, que permite acessar dados de forma extremamente rápida. 


dados = {
  "nome": "Fabio",
  "idade": 30
}

Métodos Importantes

get()


pessoa = {"nome": "Ana"}

print(pessoa.get("nome"))
print(pessoa.get("idade"))

keys()


print(pessoa.keys())

values()


print(pessoa.values())

items()


for chave, valor in pessoa.items():
  print(chave, valor)

Exemplo Prático


texto = "python é poderoso e python é simples"

palavras = texto.split()

contador = {}

for palavra in palavras:

  if palavra in contador:
      contador[palavra] += 1
  else:
      contador[palavra] = 1

print(contador)

3. Funções Lambda

Funções lambda são funções anônimas e pequenas, utilizadas geralmente para operações rápidas. 


soma = lambda a, b: a + b

print(soma(3,5))

Exemplo com Ordenação


alunos = [
  ("Ana", 8),
  ("Carlos", 6),
  ("Maria", 9)
]

alunos.sort(key=lambda aluno: aluno[1])

print(alunos)

4. Geradores (Generators)

Geradores permitem produzir valores sob demanda utilizando a palavra yield. 


def gerar_numeros():

  for i in range(5):
      yield i

Utilização:


for numero in gerar_numeros():
  print(numero)

Uso com Arquivos Grandes


def ler_arquivo(nome):

  with open(nome) as f:
      for linha in f:
          yield linha

5. Closures

Uma closure ocorre quando uma função interna lembra variáveis do escopo externo. 


def multiplicador(n):

  def calcular(x):
      return x * n

  return calcular

Uso:


dobro = multiplicador(2)

print(dobro(5))

6. Decoradores

Decoradores são funções que modificam ou estendem o comportamento de outras funções. 


def meu_decorador(func):

  def wrapper():

      print("Antes da função")
      func()
      print("Depois da função")

  return wrapper

Aplicando o decorador:


@meu_decorador
def dizer_ola():
  print("Olá")

dizer_ola()

Exemplo Real: Log de Execução


def log(func):

  def wrapper(*args, **kwargs):

      print("Executando função...")
      resultado = func(*args, **kwargs)
      print("Função finalizada")

      return resultado

  return wrapper
        

@log
def soma(a,b):
  return a+b

print(soma(3,4))
