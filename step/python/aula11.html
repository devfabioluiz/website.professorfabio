Agentes de IA com CrewAI: Analista de Vaga Automático

Tópicos abordados nessa aula: Agentes de IA, CrewAI (Agents, Tasks, Crew), Groq API, Web Scraping, Prompt Engineering, Multi-Agent Systems.

Objetivo da aula

Nesta aula, vamos entrar no mundo dos agentes de IA: sistemas que não apenas respondem perguntas, mas tomam decisões, usam ferramentas e colaboram entre si para completar tarefas complexas.

Projeto: Vamos construir um Analista de Vaga Automático. Você cola a URL de uma vaga real, fornece seu currículo, e uma equipe de 3 agentes (Scraper → Analyst → Strategist) entrega: relatório de match, cover letter personalizada e plano de estudos.

Conteúdos da aula

  • O que são agentes de IA e como diferem de chatbots
  • Multi-agente: quando dividir o trabalho entre especialistas
  • CrewAI: Agent (role, goal, backstory), Task, Crew
  • Processo sequential — cada agente recebe o output do anterior
  • Groq como LLM gratuito — llama-3.3-70b, sem GPU
  • Ferramentas: ScrapeWebsiteTool, FileReadTool
  • Prompt Engineering por agente — backstory molda tom e precisão

1. Agentes de IA vs Chatbots

Um chatbot (como o ChatGPT) segue o padrão: pergunta → resposta. Um agente vai além:

Chatbot Agente de IA
Recebe pergunta, devolve resposta Recebe objetivo, decide próximos passos
Não usa ferramentas externas Usa ferramentas (scraping, arquivos, APIs)
Atua sozinho Pode atuar em equipe (multi-agente)
Sem memória entre interações Mantém contexto e delega entre agentes
Exemplo: "O que é Python?" Exemplo: "Analise esta vaga, compare com meu CV e gere uma carta"

Por que multi-agente?

Tarefas complexas se beneficiam de especialização. Em vez de um agente tentar fazer tudo, você tem: 


Tarefa: "Analisar vaga de Desenvolvedor Python no LinkedIn"

     ┌─────────────────┐
     │   Job Scraper    │  → Especialista em extrair informações de páginas web
     │   Agent          │
     └────────┬────────┘
              │ output: requisitos, stack, senioridade
              ▼
     ┌─────────────────┐
     │   CV Analyst     │  → Especialista em comparar skills e mapear gaps
     │   Agent          │
     └────────┬────────┘
              │ output: matriz de match, gaps identificados
              ▼
     ┌─────────────────┐
     │ Career Strategist│  → Especialista em carreira: carta + plano de estudo
     │   Agent          │
     └────────┬────────┘
              │ output: cover_letter.md + study_plan.md
              ▼
         Arquivos prontos para usar!

2. CrewAI — O Framework

CrewAI é um framework Python para orquestrar agentes de IA colaborativos. Os 3 pilares são:

Agent

Define quem é o agente: seu papel, objetivo, personalidade e ferramentas. 


from crewai import Agent

job_scraper = Agent(
    role="Tech Recruiter",
    goal="Extrair todos os requisitos e detalhes da vaga a partir da URL fornecida",
    backstory=(
        "Você é um recrutador tech sênior com 15 anos de experiência. "
        "Sua especialidade é ler descrições de vagas e extrair informações "
        "estruturadas: stack tecnológica, nível de senioridade, requisitos "
        "obrigatórios e diferenciais, benefícios e faixa salarial. "
        "Você é meticuloso e nunca perde um detalhe importante."
    ),
    tools=[ScrapeWebsiteTool()],
    llm=llm,
    verbose=True,
)

Task

Define o que fazer: descrição, saída esperada, qual agente executa. 


from crewai import Task

scrape_task = Task(
    description=(
        "Acesse a URL da vaga {url_vaga} e extraia todas as informações relevantes. "
        "Organize em: título da vaga, empresa, stack tecnológica, requisitos obrigatórios, "
        "requisitos diferenciais, nível de senioridade, benefícios e faixa salarial (se disponível)."
    ),
    expected_output=(
        "Um relatório estruturado em markdown com todas as informações da vaga, "
        "organizado por seções claras."
    ),
    agent=job_scraper,
)

Crew

Orquestra os agentes e tarefas, definindo o fluxo de execução.


from crewai import Crew

crew = Crew(
    agents=[job_scraper, cv_analyst, career_strategist],
    tasks=[scrape_task, analyze_task, strategize_task],
    process=Process.sequential,   # um após o outro
    verbose=True,
)

resultado = crew.kickoff()

3. Groq — LLM Gratuito e Rápido

O Groq oferece acesso gratuito a modelos de linguagem de alta qualidade via API, com latência muito baixa. Ideal para projetos que não exigem GPU local.

Característica Groq Ollama (local)
GPU necessária Não (roda na nuvem) Sim (roda na sua máquina)
Custo Grátis (limite generoso) Grátis
Modelos disponíveis llama-3.3-70b, qwen-3.6, deepseek Depende do download
Velocidade Muito rápida (hardware dedicado) Depende da sua GPU
Internet Necessária Não necessária

Configurando o Groq

1. Acesse console.groq.com

2. Crie uma conta gratuita (pode usar login Google/GitHub)

3. Vá em API Keys e crie uma chave

4. Guarde a chave — ela só aparece uma vez

4. Ferramentas (Tools)

Ferramentas são funções que os agentes podem chamar para interagir com o mundo real. O CrewAI já vem com várias prontas.

ScrapeWebsiteTool

Raspa o conteúdo de qualquer página web. O agente vai usar isso para ler a vaga. 


from crewai_tools import ScrapeWebsiteTool

scrape_tool = ScrapeWebsiteTool()
# O agente pode chamar: scrape_tool.run(url="https://...")

FileReadTool

Lê arquivos do sistema. O agente vai usar para ler o currículo do aluno. 


from crewai_tools import FileReadTool

leitor_cv = FileReadTool(file_path="meu_cv.txt")
# O agente pode chamar: leitor_cv.run()

import os
from dotenv import load_dotenv
from crewai import Agent, LLM
from crewai_tools import ScrapeWebsiteTool, FileReadTool

load_dotenv()

llm = LLM(
    model="llama-3.3-70b-versatile",
    base_url="https://api.groq.com/openai/v1",
    api_key=os.getenv("GROQ_API_KEY"),
    temperature=0.3,
    provider="openai",
)

# ============================================================
# FERRAMENTAS
# ============================================================
scrape_tool = ScrapeWebsiteTool()
leitor_cv = FileReadTool(file_path="meu_cv.txt")

# ============================================================
# AGENTES
# ============================================================
job_scraper = Agent(
    role="Tech Recruiter Sênior",
    goal="Extrair e estruturar todas as informações relevantes da vaga a partir da URL fornecida",
    backstory=(
        "Você é um recrutador tech com 15 anos de experiência em grandes empresas "
        "de tecnologia. Sua especialidade é analisar descrições de vagas e extrair "
        "informações de forma estruturada. Você identifica padrões em requisitos, "
        "diferencia must-have de nice-to-have, e entende profundamente stacks "
        "tecnológicas modernas. Você é detalhista e nunca deixa passar um requisito "
        "importante, por mais sutil que esteja na descrição."
    ),
    tools=[scrape_tool],
    llm=llm,
    verbose=True,
)

cv_analyst = Agent(
    role="Analista de Currículos e Match de Skills",
    goal="Comparar o currículo do candidato com os requisitos da vaga e gerar uma matriz de compatibilidade",
    backstory=(
        "Você é um analista de RH especializado em matching de currículos com vagas. "
        "Com 10 anos de experiência, você é capaz de identificar rapidamente gaps de "
        "habilidades, pontos fortes e fracos de um candidato em relação a uma vaga. "
        "Você é justo e transparente: aponta tanto alinhamentos quanto lacunas. "
        "Seu relatório é sempre estruturado em: skills que o candidato TEM, skills "
        "que o candidato TEM PARCIALMENTE, e skills que o candidato NÃO TEM."
    ),
    tools=[leitor_cv],
    llm=llm,
    verbose=True,
)

career_strategist = Agent(
    role="Coach de Carreira e Estrategista de Desenvolvimento Profissional",
    goal="Criar uma cover letter personalizada e um plano de estudos de 30 dias baseado nos gaps identificados",
    backstory=(
        "Você é um coach de carreira renomado, com centenas de profissionais "
        "recolocados em empresas como Google, Nubank, iFood e outras. Sua especialidade "
        "é criar cartas de apresentação que destacam os pontos fortes do candidato "
        "enquanto abordam gaps de forma estratégica. Você também é excelente em criar "
        "roadmaps de aprendizado realistas, priorizando o que traz mais retorno rápido. "
        "Seu tom é motivador mas realista, profissional mas pessoal."
    ),
    llm=llm,
    verbose=True,
)

Passo 6 — Criar o orchestrator (main.py)

Arquivo: main.py


import os
from datetime import datetime
from crewai import Crew, Process
from agents import job_scraper, cv_analyst, career_strategist
from tasks import scrape_task, analyze_task, strategize_task

# ============================================================
# CONFIGURAÇÃO
# ============================================================
# Cole aqui a URL da vaga que quer analisar
URL_VAGA = input("Cole a URL da vaga: ").strip()

if not URL_VAGA:
    print("❌ Nenhuma URL fornecida. Usando URL de exemplo...")
    URL_VAGA = "https://exemplo.com/vaga-python"  # substitua por uma real

# ============================================================
# MONTAR A CREW
# ============================================================
crew = Crew(
    agents=[job_scraper, cv_analyst, career_strategist],
    tasks=[scrape_task, analyze_task, strategize_task],
    process=Process.sequential,
    verbose=True,
)

# ============================================================
# EXECUTAR
# ============================================================
print("\n" + "=" * 60)
print("  🚀 INICIANDO ANÁLISE DA VAGA")
print("=" * 60)
print(f"  URL: {URL_VAGA}")
print(f"  Modelo: llama-3.3-70b (Groq)")
print("=" * 60 + "\n")

resultado = crew.kickoff(inputs={"url_vaga": URL_VAGA})

# ============================================================
# SALVAR RESULTADOS
# ============================================================
timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")

# Salvar resultado completo
os.makedirs("output", exist_ok=True)
with open(f"output/resultado_completo_{timestamp}.md", "w", encoding="utf-8") as f:
    f.write(f"# Análise de Vaga — {timestamp}\n\n")
    f.write(f"**URL:** {URL_VAGA}\n\n")
    f.write("---\n\n")
    f.write(str(resultado))

print("\n" + "=" * 60)
print("  ✅ ANÁLISE CONCLUÍDA!")
print(f"  Relatório salvo em: output/resultado_completo_{timestamp}.md")
print("=" * 60)

Passo 7 — Atualizar a tasks.py e executar o projeto

Dentro de tasks.pyinsira o código abaixo:


from crewai import Task
from agents import job_scraper, cv_analyst, career_strategist

# ============================================================
# TAREFAS
# ============================================================
scrape_task = Task(
    description=(
        "1. Acesse a URL da vaga: {url_vaga}\n"
        "2. Extraia TODO o conteúdo relevante da descrição da vaga.\n"
        "3. Organize as informações no seguinte formato estruturado:\n"
        "   - Título da vaga e nome da empresa\n"
        "   - Stack tecnológica (linguagens, frameworks, ferramentas)\n"
        "   - Requisitos obrigatórios (must-have)\n"
        "   - Requisitos diferenciais (nice-to-have)\n"
        "   - Nível de senioridade exigido\n"
        "   - Tipo de contrato (CLT, PJ, remoto, híbrido, presencial)\n"
        "   - Benefícios oferecidos\n"
        "   - Faixa salarial (se disponível)\n"
        "4. Se a página não for uma vaga válida ou não puder ser acessada, "
        "informe claramente o problema."
    ),
    expected_output=(
        "Um relatório detalhado em markdown com todas as informações da vaga, "
        "organizado por seções conforme o formato solicitado."
    ),
    agent=job_scraper,
)

analyze_task = Task(
    description=(
        "1. Leia o arquivo 'meu_cv.txt' para conhecer o perfil do candidato.\n"
        "2. Com base na análise da vaga (output da tarefa anterior), compare "
        "sistematicamente cada requisito com o currículo do candidato.\n"
        "3. Classifique cada skill em uma das três categorias:\n"
        "   ✅ TEM — o candidato possui essa skill comprovadamente\n"
        "   ⚠️ TEM PARCIALMENTE — possui conhecimento básico ou relacionado\n"
        "   ❌ NÃO TEM — não há evidência dessa skill no currículo\n"
        "4. Calcule uma porcentagem de match geral.\n"
        "5. Destaque os 3 principais pontos fortes do candidato para esta vaga.\n"
        "6. Destaque os 3 principais gaps que precisam ser endereçados."
    ),
    expected_output=(
        "Um relatório de match em markdown contendo:\n"
        "1. Tabela de compatibilidade skill por skill\n"
        "2. Porcentagem de match geral\n"
        "3. Top 3 pontos fortes\n"
        "4. Top 3 gaps críticos"
    ),
    agent=cv_analyst,
    context=[scrape_task],   # recebe o output da tarefa anterior
)

strategize_task = Task(
    description=(
        "Com base nos outputs anteriores, produza DOIS documentos separados:\n\n"
        "DOCUMENTO 1 — COVER LETTER:\n"
        "- Carta de apresentação personalizada para esta vaga específica\n"
        "- Tom profissional mas autêntico\n"
        "- Destaque 3 experiências/habilidades do candidato alinhadas com a vaga\n"
        "- Mencione 1 gap de forma estratégica (como motivação para aprender)\n"
        "- Inclua parágrafo de abertura, 2-3 parágrafos de corpo e fechamento\n\n"
        "DOCUMENTO 2 — PLANO DE ESTUDOS DE 30 DIAS:\n"
        "- Foco nos gaps críticos identificados\n"
        "- Dividido em 4 semanas com metas realistas\n"
        "- Cada semana com tópicos específicos, recursos sugeridos e mini-projeto\n"
        "- Priorize o que tem maior impacto para esta vaga\n\n"
        "Separe os dois documentos claramente com títulos em markdown."
    ),
    expected_output=(
        "Dois documentos em markdown:\n"
        "1. Cover letter completa e pronta para enviar\n"
        "2. Plano de estudos de 30 dias estruturado por semanas"
    ),
    agent=career_strategist,
    context=[scrape_task, analyze_task],
)

Resumo da aula

Conceito Classe/Função Para que serve
Agente Agent(role, goal, backstory, tools, llm) Define quem é o agente, seu papel e personalidade
Tarefa Task(description, expected_output, agent, context) Define o que fazer e qual a saída esperada
Crew Crew(agents, tasks, process) Orquestra agentes e tarefas
Processo sequencial Process.sequential Cada agente recebe o output do anterior
LLM Groq LLM(model="groq/llama-3.3-70b-versatile") Modelo de linguagem gratuito via API
Web Scraping ScrapeWebsiteTool() Agente lê páginas web automaticamente
Leitura de arquivos FileReadTool(file_path="...") Agente lê arquivos locais (CV, etc.)
Prompt Engineering Backstory + description + expected_output Quanto mais específico, melhor o resultado

Dever de Casa

1. Execute com uma vaga real do seu interesse

Encontre uma vaga real no LinkedIn, Programathor ou similar que corresponda à sua área de interesse. Execute o analista com essa vaga e seu currículo real. Leia os 3 arquivos gerados e avalie se a cover letter está convincente.

Dicas:
  • Prefira vagas com descrições detalhadas (mais de 5 requisitos listados)
  • Quanto mais completo seu currículo no meu_cv.txt, melhor o match
  • Se o scraping falhar, teste com outra URL (alguns sites bloqueiam bots)
  • Ajuste o temperature no LLM se quiser respostas mais criativas (0.7) ou mais precisas (0.1)

2. Adicione um quarto agente: Interview Coach

Crie um novo agente chamado interview_coach com a tarefa de gerar 5 perguntas prováveis de entrevista técnica baseadas na vaga, junto com sugestões de resposta. Conecte ele no fluxo sequential após o career_strategist.

Gabarito: 

# agents.py — adicione
interview_coach = Agent(
    role="Entrevistador Técnico Sênior",
    goal="Gerar perguntas de entrevista técnica personalizadas baseadas na vaga",
    backstory=(
        "Você é um engenheiro de software sênior que já conduziu mais de 500 "
        "entrevistas técnicas em big techs. Você sabe exatamente quais perguntas "
        "testam o conhecimento real do candidato para cada tecnologia listada na vaga. "
        "Para cada pergunta, você também fornece uma sugestão de como estruturar "
        "a resposta ideal."
    ),
    llm=llm,
    verbose=True,
)

# tasks.py — adicione
interview_task = Task(
    description=(
        "Com base na vaga e no perfil do candidato:\n"
        "1. Gere 5 perguntas de entrevista técnica específicas para esta vaga\n"
        "2. Para cada pergunta, explique o que o entrevistador quer avaliar\n"
        "3. Dê uma sugestão de estrutura de resposta (sem dar a resposta pronta)\n"
        "4. Foque nas tecnologias onde o candidato tem gap — são as mais prováveis de cair"
    ),
    expected_output="5 perguntas de entrevista com explicação e estrutura de resposta sugerida",
    agent=interview_coach,
    context=[scrape_task, analyze_task],
)

# main.py — atualize a crew
crew = Crew(
    agents=[job_scraper, cv_analyst, career_strategist, interview_coach],
    tasks=[scrape_task, analyze_task, strategize_task, interview_task],
    process=Process.sequential,
    verbose=True,
)
