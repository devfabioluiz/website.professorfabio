O que é o padrão MVT?

O padrão MVT (Model-View-Template) é uma variação do MVC muito utilizada em frameworks como Django.

Objetivos do MVT

  • Separar responsabilidades
  • Organizar o código
  • Facilitar manutenção

Model

O que é? Representa os dados

Responsabilidade: Gerenciar banco e regras

View

O que é? Lógica da aplicação

Responsabilidade: Receber requisição e processar

Template

O que é? Interface visual

Responsabilidade: Exibir HTML

Fluxo MVT

  • Usuário faz requisição
  • View recebe
  • View chama Model
  • View envia dados para Template
  • Template renderiza resposta

Passo 6 - View

Arquivo: app/views/usuario_view.py


from app.models import usuario_model
from app.templates import usuario_template


def listar_view():
    dados = usuario_model.listar()
    return usuario_template.render(dados)


def criar_view(nome):
    usuario_model.criar(nome)
    return {"msg": "criado"}

Passo 7 - FastAPI

Arquivo: main.py


from fastapi import FastAPI
from app.views import usuario_view

app = FastAPI()

@app.get("/usuarios")
def listar():
    return usuario_view.listar_view()

@app.post("/usuarios")
def criar(nome: str):
    return usuario_view.criar_view(nome)
