Padrão MVT (Model-View-Template) com Django

O que é o padrão MVT?

O padrão MVT é usado pelo Django para organizar aplicações. Ele é similar ao MVC, mas com pequenas diferenças:

  • Model: Representa os dados e regras de negócio.
  • View: Contém a lógica da aplicação, recebe requisições e processa dados.
  • Template: Define a interface visual, renderizando HTML.

Fluxo MVT no Django

  1. Usuário faz uma requisição HTTP
  2. Django direciona para a View
  3. View consulta os Models se necessário
  4. View envia os dados para um Template
  5. Template renderiza a resposta HTML para o usuário

Passo 3 — Model

Arquivo: usuarios/models.py


from django.db import models

class Usuario(models.Model):
    nome = models.CharField(max_length=100)

    def __str__(self):
        return self.nome

Passo 4 — Serializer (para API REST)

Arquivo: usuarios/serializers.py


from rest_framework import serializers
from .models import Usuario

class UsuarioSerializer(serializers.ModelSerializer):
    class Meta:
        model = Usuario
        fields = '__all__'

Passo 5 — View

Arquivo: usuarios/views.py


from django.shortcuts import render
from rest_framework.decorators import api_view
from rest_framework.response import Response
from .models import Usuario
from .serializers import UsuarioSerializer

# API: listar usuários
@api_view(['GET'])
def listar_usuarios(request):
    usuarios = Usuario.objects.all()
    serializer = UsuarioSerializer(usuarios, many=True)
    return Response(serializer.data)

# API: criar usuário
@api_view(['POST'])
def criar_usuario(request):
    serializer = UsuarioSerializer(data=request.data)
    if serializer.is_valid():
        serializer.save()
        return Response({"msg": "Usuario criado"})
    return Response(serializer.errors)

Passo 7 — View para template

Arquivo: usuarios/views.py


def listar_template(request):
    usuarios = Usuario.objects.all()
    return render(request, 'usuarios/lista.html', {'usuarios': usuarios})

Passo 8 — URLs

Arquivo: usuarios/urls.py


from django.urls import path
from . import views

urlpatterns = [
    path('api/usuarios/', views.listar_usuarios),
    path('api/usuarios/criar/', views.criar_usuario),
    path('usuarios/', views.listar_template),
]

Em projeto_django_mvt/urls.py, adicione:


from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('usuarios.urls')),
]
