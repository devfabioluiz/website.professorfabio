Criar a tabela "Favorito" (Model)

Abra o arquivo models.py e cole TODO este código:

📁 favoritos/models.py
from django.db import models

class Favorito(models.Model):
    CATEGORIAS = [
        ('animal', 'Animal'),
        ('jogo', 'Jogo'),
        ('personagem', 'Personagem'),
        ('comida', 'Comida'),
        ('outro', 'Outro'),
    ]

    nome = models.CharField(max_length=100)
    categoria = models.CharField(max_length=20, choices=CATEGORIAS)
    descricao = models.TextField(blank=True)

    def __str__(self):
        return self.nome
📖 O que esse código faz: Cria a "ficha" do favorito. Cada favorito tem: nome (texto), categoria (escolhe entre: Animal, Jogo, Personagem, Comida, Outro), descrição (texto maior, opcional).
✅ Passo 3 concluído! A tabela "Favorito" está pronta.
4

Criar o formulário

Crie um arquivo novo chamado forms.py dentro da pasta favoritos/ e cole:

📁 favoritos/forms.py (criar novo)
from django import forms
from .models import Favorito

class FavoritoForm(forms.ModelForm):
    class Meta:
        model = Favorito
        fields = ['nome', 'categoria', 'descricao']
📖 O que isso faz: Cria um formulário automático baseado no model Favorito. O Django já sabe os campos (nome, categoria, descricao) e cria os inputs pra gente.
✅ Passo 4 concluído! O formulário está pronto.
5

Criar as funções (Views)

Abra o arquivo views.py e substitua TODO o conteúdo por este:

📁 favoritos/views.py
from django.shortcuts import render, redirect
from .models import Favorito
from .forms import FavoritoForm

def listar(request):
    favoritos = Favorito.objects.all()
    return render(request, 'favoritos/lista.html', {'favoritos': favoritos})

def adicionar(request):
    if request.method == 'POST':
        form = FavoritoForm(request.POST)
        if form.is_valid():
            form.save()
            return redirect('listar')
    else:
        form = FavoritoForm()
    return render(request, 'favoritos/adicionar.html', {'form': form})
📖 O que esse código faz: Duas funções:
listar → busca todos os favoritos do banco e manda pro template.
adicionar → mostra o formulário vazio. Se o usuário preencher e salvar, guarda no banco.
✅ Passo 5 concluído! As funções estão prontas.
6

Criar as URLs (endereços)

Crie um arquivo urls.py dentro da pasta favoritos/ e cole:

📁 favoritos/urls.py (criar novo)
from django.urls import path
from . import views

urlpatterns = [
    path('', views.listar, name='listar'),
    path('adicionar/', views.adicionar, name='adicionar'),
]
📖 O que isso faz: Define os endereços:
/ → mostra a lista de favoritos
/adicionar/ → mostra o formulário para adicionar

Agora abra o arquivo urls.py principal e atualize:

📁 projeto_django_mvt/urls.py
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('favoritos.urls')),
]
📖 O que isso faz: Diz pro Django: "quando alguém acessar o site, usa as URLs do app favoritos".
✅ Passo 6 concluído! Os caminhos estão configurados.

8

Rodar e ver seus favoritos!

Último passo! Cria as tabelas no banco e inicia o servidor:

python manage.py makemigrations
python manage.py migrate
python manage.py runserver
📖 O que esses comandos fazem:
makemigrations → prepara a criação da tabela "Favorito"
migrate → cria de fato a tabela no banco
runserver → liga o servidor local

Agora abra o navegador e acesse:

http://localhost:8000/

Para adicionar favoritos, clique no botão "+ Adicionar Favorito" e preencha o formulário.

✅ Passo 8 concluído! Seu catálogo está no ar!
