1. Tuplas

Tuplas são coleções ordenadas e imutáveis. Isso significa que seus valores podem ser acessados por índice, mas não podem ser alterados depois da criação.

Características das tuplas

  • São imutáveis
  • Permitem acesso por índice
  • Podem armazenar valores de tipos diferentes
  • São úteis para dados fixos
  • Podem ser desempacotadas em variáveis

Exemplo básico


coordenada = (10, 20)

print(coordenada[0])
print(coordenada[1])

Exemplo com desempacotamento


usuario = ("Fabio", "Desenvolvedor", "Admin")

nome, cargo, perfil = usuario

print(nome)
print(cargo)
print(perfil)

Exemplo prático: retorno de função

Uma função pode retornar uma tupla quando precisamos devolver mais de uma informação. 


def calcular_total(preco, quantidade):
    total = preco * quantidade
    mensagem = "Compra calculada com sucesso"
    return total, mensagem

valor, texto = calcular_total(50, 3)

print(valor)
print(texto)

2. Dicionários

Dicionários são estruturas que armazenam dados no formato chave e valor. Eles são muito usados para representar objetos, usuários, produtos, respostas de APIs e configurações de sistemas.

Mapas associativos

Um dicionário funciona como um mapa associativo porque associa uma chave a um valor. 


usuario = {
    "nome": "Fabio",
    "idade": 32,
    "profissao": "Desenvolvedor"
}

print(usuario["nome"])
print(usuario["profissao"])

Tabelas hash

Internamente, dicionários utilizam uma estrutura baseada em tabela hash. Isso permite localizar valores de forma rápida a partir de uma chave. 


usuarios = {
    1: "Ana",
    2: "Carlos",
    3: "Marina"
}

print(usuarios[2])

Métodos principais

Método Descrição
get() Busca um valor sem gerar erro caso a chave não exista
keys() Retorna as chaves do dicionário
values() Retorna os valores do dicionário
items() Retorna pares de chave e valor
update() Atualiza ou adiciona dados
pop() Remove uma chave e retorna o valor removido

Exemplo com métodos


produto = {
    "nome": "Notebook",
    "preco": 3500,
    "estoque": 10
}

print(produto.get("nome"))
print(produto.get("categoria", "Sem categoria"))

produto.update({"preco": 3200, "categoria": "Informática"})

for chave, valor in produto.items():
    print(chave, valor)

estoque_removido = produto.pop("estoque")
print(estoque_removido)
print(produto)

Exemplo prático: controle de estoque


estoque = {
    "notebook": 10,
    "mouse": 25,
    "teclado": 15
}

produto = "mouse"

if estoque.get(produto, 0) > 0:
    print("Produto disponível")
else:
    print("Produto indisponível")

3. Funções Lambda

Funções lambda são funções anônimas, geralmente usadas para operações simples e rápidas. Elas são comuns em ordenações, filtros e transformações de listas.

Função comum


def dobro(numero):
    return numero * 2

print(dobro(5))

Mesma lógica com lambda


dobro = lambda numero: numero * 2

print(dobro(5))

Exemplo com map()


numeros = [1, 2, 3, 4]

dobrados = list(map(lambda numero: numero * 2, numeros))

print(dobrados)

Exemplo com filter()


numeros = [1, 2, 3, 4, 5, 6]

pares = list(filter(lambda numero: numero % 2 == 0, numeros))

print(pares)

Exemplo prático: ordenar usuários por idade


usuarios = [
    {"nome": "Ana", "idade": 25},
    {"nome": "Carlos", "idade": 20},
    {"nome": "Marina", "idade": 30}
]

usuarios_ordenados = sorted(usuarios, key=lambda usuario: usuario["idade"])

print(usuarios_ordenados)

4. Geradores

Geradores são funções que produzem valores sob demanda. Em vez de retornar todos os dados de uma vez, eles usam a palavra-chave yield para entregar um valor por vez.

Exemplo básico


def contador():
    yield 1
    yield 2
    yield 3

for numero in contador():
    print(numero)

Exemplo prático: gerar números até um limite


def gerar_numeros(limite):
    numero = 1

    while numero <= limite:
        yield numero
        numero += 1

for n in gerar_numeros(5):
    print(n)

Comparação com lista

Uma lista cria todos os valores na memória. Um gerador cria os valores aos poucos. 


lista = [numero for numero in range(1000000)]
gerador = (numero for numero in range(1000000))

print(type(lista))
print(type(gerador))

5. Closures

Closure acontece quando uma função interna guarda acesso às variáveis da função externa, mesmo depois que a função externa terminou sua execução.

Exemplo básico


def criar_multiplicador(fator):
    def multiplicar(numero):
        return numero * fator

    return multiplicar

dobrar = criar_multiplicador(2)
triplicar = criar_multiplicador(3)

print(dobrar(10))
print(triplicar(10))

Exemplo prático: validador de senha


def criar_validador(tamanho_minimo):
    def validar(texto):
        return len(texto) >= tamanho_minimo

    return validar

validar_senha = criar_validador(8)

print(validar_senha("abc"))
print(validar_senha("abc12345"))

6. Decoradores

Decoradores permitem modificar ou adicionar comportamento a uma função sem alterar diretamente o código principal dela. Eles são muito usados em APIs, autenticação, logs, permissões e validações.

Exemplo básico


def meu_decorador(funcao):
    def wrapper():
        print("Antes da função")
        funcao()
        print("Depois da função")

    return wrapper

@meu_decorador
def saudacao():
    print("Olá!")

saudacao()

Decorador com argumentos


def logar_execucao(funcao):
    def wrapper(*args, **kwargs):
        print(f"Executando a função {funcao.__name__}")
        resultado = funcao(*args, **kwargs)
        print("Execução finalizada")
        return resultado

    return wrapper

@logar_execucao
def somar(a, b):
    return a + b

resultado = somar(10, 5)
print(resultado)

Exemplo prático: verificar login


def exige_login(funcao):
    def wrapper(usuario):
        if not usuario.get("logado"):
            return "Acesso negado"

        return funcao(usuario)

    return wrapper

@exige_login
def acessar_painel(usuario):
    return f"Bem-vindo, {usuario['nome']}"

usuario1 = {
    "nome": "Fabio",
    "logado": True
}

usuario2 = {
    "nome": "Ana",
    "logado": False
}

print(acessar_painel(usuario1))
print(acessar_painel(usuario2))

Dever de Casa

1. Crie uma tupla com os dados de um produto

A tupla deve conter nome, preço e quantidade. Depois, desempacote os valores em variáveis e exiba o total da compra.

Gabarito: 

produto = ("Notebook", 3500, 2)

nome, preco, quantidade = produto

total = preco * quantidade

print(f"Produto: {nome}")
print(f"Total: R$ {total}")

2. Crie um dicionário para representar um aluno

O dicionário deve ter nome, notas e situação. Calcule a média e informe se o aluno foi aprovado ou reprovado.

Gabarito: 

aluno = {
    "nome": "João",
    "notas": [8, 7, 9]
}

media = sum(aluno["notas"]) / len(aluno["notas"])

if media >= 7:
    aluno["situacao"] = "Aprovado"
else:
    aluno["situacao"] = "Reprovado"

print(aluno)

3. Use lambda para filtrar produtos caros

Crie uma lista de produtos e filtre apenas os produtos com preço maior que 1000.

Gabarito: 

produtos = [
    {"nome": "Mouse", "preco": 80},
    {"nome": "Notebook", "preco": 3500},
    {"nome": "Monitor", "preco": 1200}
]

produtos_caros = list(filter(lambda produto: produto["preco"] > 1000, produtos))

print(produtos_caros)

4. Crie um gerador de números pares

O gerador deve receber um limite e retornar apenas números pares até esse limite.

Gabarito: 

def gerar_pares(limite):
    numero = 0

    while numero <= limite:
        yield numero
        numero += 2

for par in gerar_pares(10):
    print(par)

5. Crie uma closure para aplicar desconto

A função externa deve receber o percentual de desconto. A função interna deve receber o preço e retornar o valor com desconto.

Gabarito: 

def criar_desconto(percentual):
    def aplicar(preco):
        return preco - (preco * percentual / 100)

    return aplicar

desconto_10 = criar_desconto(10)

print(desconto_10(1000))

6. Crie um decorador de log

O decorador deve mostrar uma mensagem antes e depois da execução de uma função.

Gabarito: 

def log(funcao):
    def wrapper(*args, **kwargs):
        print("Iniciando execução")
        resultado = funcao(*args, **kwargs)
        print("Finalizando execução")
        return resultado

    return wrapper

@log
def cadastrar_usuario(nome):
    print(f"Usuário {nome} cadastrado")

cadastrar_usuario("Fabio")
