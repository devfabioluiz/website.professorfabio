Banco de Dados

Introdução ao MySQL – Tabelas, Chaves e DML

1. Criando Tabelas

Para criar uma tabela no MySQL utilizamos o comando CREATE TABLE. Nele definimos colunas, tipos de dados e restrições.

Exemplo – criando a tabela "clientes"

CREATE TABLE clientes (
  id INT AUTO_INCREMENT,
  nome VARCHAR(100) NOT NULL,
  email VARCHAR(150),
  PRIMARY KEY (id)
);

2. Excluir Tabela

Para excluir uma tabela permanentemente do banco utilizamos o comando DROP TABLE.

Exemplo – excluindo tabela

DROP TABLE clientes;

3. Definir Chave Primária (PRIMARY KEY)

A chave primária identifica cada linha de forma única e não pode se repetir. Pode ser criada junto com a tabela ou adicionada depois.

Exemplo – chave primária na criação

CREATE TABLE produtos (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nome VARCHAR(100),
  preco DECIMAL(10,2)
);
Exemplo – adicionando chave primária depois

ALTER TABLE produtos
ADD PRIMARY KEY (id);

4. Definir Chave Estrangeira (FOREIGN KEY)

A chave estrangeira relaciona duas tabelas, garantindo integridade referencial.

Exemplo – criando uma tabela com chave estrangeira

CREATE TABLE pedidos (
  id INT AUTO_INCREMENT PRIMARY KEY,
  cliente_id INT,
  valor DECIMAL(10,2),
  FOREIGN KEY (cliente_id) REFERENCES clientes(id)
);

5. Comandos Básicos DML (INSERT, SELECT, UPDATE, DELETE)

Estes comandos manipulam dados dentro das tabelas.

INSERT – inserir dados

INSERT INTO clientes (nome, email)
VALUES ('João Silva', 'joao@email.com');
SELECT – consultar dados

SELECT * FROM clientes;
SELECT nome, email FROM clientes;
UPDATE – atualizar dados

UPDATE clientes
SET email = 'novoemail@email.com'
WHERE id = 1;
DELETE – excluir dados

DELETE FROM clientes
WHERE id = 1;

6. Criar e Selecionar Banco de Dados


CREATE DATABASE meu_banco;
USE meu_banco;
