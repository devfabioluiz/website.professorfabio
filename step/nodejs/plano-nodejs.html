Curso de Desenvolvimento de Aplicações Web com Node.js

Versão 1.1.0

Carga horária total – 38 horas (19 aulas)

Objetivo do Curso

Capacitar o estudante no desenvolvimento de soluções backend utilizando a plataforma Node.js, com ênfase na escolha adequada de mecanismos e estruturas para resolver problemas típicos do desenvolvimento web.

Ao Final do Curso

  • Compreender o uso do gerenciador de pacotes
  • Personalizar o Node.js para diferentes sistemas operacionais
  • Interagir com a interface REPL
  • Compreender a arquitetura orientada a eventos
  • Utilizar arquivos, threads e buffers
  • Diferenciar chamadas síncronas e assíncronas
  • Compreender mecanismos de escalabilidade
  • Desenvolver soluções Node.js no cliente e no servidor
  • Aplicar princípios de modelagem relacional
  • Utilizar o framework Express
  • Implementar roteamento
  • Processar dados de formulários
  • Utilizar cookies e sessões
  • Interagir com bancos de dados
  • Compreender a arquitetura REST
  • Criar serviços RESTful
