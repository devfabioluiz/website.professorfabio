Iniciando nosso projeto Final

Node.js

Projeto Final: API REST completa com Express, MongoDB, Autenticação JWT e CRUD de produtos.

O Projeto Final

Neste projeto final você criará uma API REST completa que servirá como backend para os projetos finais de React e Angular.

A API deve ser criada dentro de uma branch chamada projeto-final no repositório projeto-node-step.

A API terá os seguintes recursos:

  • Produtos — CRUD completo (listar, obter, criar, atualizar, deletar)
  • Autenticação — registro de usuário, login com JWT e perfil do usuário logado
  • Proteção de rotas — endpoints administrativos protegidos por JWT e verificação de permissão (admin)

Resumo dos Endpoints

Método Rota Autenticação Descrição
GET / Pública Informações da API e lista de endpoints
GET /produtos Pública Listar todos os produtos
GET /produtos/:id Pública Obter um produto pelo ID
POST /produtos JWT (admin) Criar um novo produto
PUT /produtos/:id JWT (admin) Atualizar um produto
DELETE /produtos/:id JWT (admin) Remover um produto
POST /registrar Pública Registrar novo usuário
POST /login Pública Fazer login e receber token JWT
GET /me JWT Obter dados do usuário logado
