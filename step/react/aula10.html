Context API (hook useContext)

A Context API no React é uma ferramenta que permite compartilhar valores e estados entre componentes sem a necessidade de passar props manualmente por toda a árvore de componentes. Ela é ideal para gerenciar dados globais, como temas, autenticação ou configurações de idioma. O contexto é criado usando React.createContext(), e seus valores são fornecidos aos componentes filhos através do Provider. Os componentes que precisam acessar esses valores utilizam o hook useContext. A Context API simplifica a gestão de estados globais em aplicações React, eliminando a complexidade de "prop drilling". 


import React, { createContext, useContext } from 'react';

const TemaContext = createContext('claro');

function Painel() {
  const tema = useContext(TemaContext);
  return <div>Tema atual: {tema}</div>;
}

function App() {
  return (
    <TemaContext.Provider value="escuro">
      <Painel />
    </TemaContext.Provider>
  );
}

Redux

Redux e uma ferramenta popular para gerenciamento de estado em aplicações React. Ele proporciona uma estrutura de dados unificada que pode ser facilmente compartilhada entre componentes e acionados por ações.

Exemplo
              
import React from 'react';
import { createStore } from 'redux';

const initialState = { count: 0 };  
const reducer = (state = initialState, action) => {
  switch (action.type) {
    case 'INCREMENT':
      return { ...state, count: state.count + 1 };
    case 'DECREMENT':
      return { ...state, count: state.count - 1 };
    default:
      return state;
  }
};

const store = createStore(reducer);

const Contador = () => {
  const state = store.getState();
  return (
    <>
      <p>Contagem: {state.count}</p>
      <button onClick={() => store.dispatch({ type: 'INCREMENT' })}>+</button>
      <button onClick={() => store.dispatch({ type: 'DECREMENT' })}>-</button>
    </>
  );
};

export default Contador;

useReducer

Hook para gerenciamento de estado complexo, similar ao Redux, com dispatch e reducer.

Exemplo

import React, { useReducer } from 'react';

const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

function Contador() {
  const [state, dispatch] = useReducer(reducer, initialState);

  return (
    <>;
      <p>;Contagem: {state.count}</p>;
      <button onClick={() =>; dispatch({ type: 'increment' })}>;+</button>;
      <button onClick={() =>; dispatch({ type: 'decrement' })}>;-</button>;
    </>;
  );
}
