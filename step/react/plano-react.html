Usando o Framework React

Versão 3.1.0

Duração do Curso – 30 horas (15 dias)

Objetivo do curso

Treinar o aluno no desenvolvimento de scripts do lado do cliente usando o framework React. Ensinar como escolher os mecanismos e construções corretas para resolver tarefas específicas.

Ao final do curso, o aluno será capaz de:

  • Entender as nuances da implementação de aplicações React
  • Criar componentes
  • Entender a diferença entre componentes de classe e funcionais
  • Trabalhar com estado
  • Usar hooks
  • Configurar roteamento
  • Interagir com o servidor
  • Usar Redux
  • Trabalhar com formulários

Critérios de avaliação

Após a conclusão deste curso, o aluno completará uma tarefa prática e um exame teórico com base no material do curso. Para ser admitido ao exame, todas as tarefas de casa e práticas devem ser concluídas. A tarefa prática deve abranger o máximo de material de várias seções do curso.
