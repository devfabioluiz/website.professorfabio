Teoria de Banco de Dados

Versão 2.1.0

Duração do Curso – 22 horas (11 aulas)

Objetivo do Curso

Treinar o aluno nos fundamentos da teoria de banco de dados. Explicar os princípios de construção de bancos de dados. Ensinar a usar a linguagem de consulta estruturada SQL para interagir com os dados. Explorar várias construções da linguagem de consulta SQL para a construção correta de consultas otimizadas.

Ao Final do Curso

  • Projetar bancos de dados
  • Aplicar formas normais para normalizar tabelas
  • Interagir com dados usando SQL
  • Criar consultas e subconsultas multitabulares
  • Usar funções de agregação
  • Entender os princípios de aplicação de várias construções SQL
  • Desenvolver procedimentos armazenados, triggers e funções definidas pelo usuário
  • Usar funções de janela (window functions)
  • Criar e modificar estruturas de banco de dados usando mecanismos DDL
  • Aplicar mecanismos de backup e recuperação
  • Compreender os fundamentos da segurança no PostgreSQL

Avaliação do Curso

Após a conclusão deste curso, o aluno completará uma tarefa prática e um exame teórico com base no material do curso. Para ser admitido ao exame, todas as tarefas de casa e práticas devem ser concluídas. A tarefa prática deve abranger o máximo de material de várias seções do curso.

Recomenda-se o uso do PostgreSQL como o SGBD para este curso. No entanto, os MySQL ou Mariadb podem ser considerados como uma alternativa.
