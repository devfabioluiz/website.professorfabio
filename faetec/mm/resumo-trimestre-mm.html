Resumo do Trimestre

1. Processador

O processador (ou CPU) é o "cérebro" do computador, responsável por executar as instruções dos programas. Ele realiza operações como cálculos, movimentação de dados e decisões lógicas.

Como Funciona o Processador

O processador realiza o ciclo de instrução, que se divide em três etapas principais:

  • Busca: O processador pega a instrução do programa armazenada na memória.
  • Decodificação: O processador interpreta a instrução.
  • Execução: O processador executa a operação.

Componentes Eletrônicos do Processador

  • ALU (Unidade Aritmético-Logica): Realiza operações matemáticas e lógicas.
  • Unidade de Controle: Coordena as operações dentro do processador.
  • Registradores: Memórias temporárias para armazenar dados.
  • Cache: Memória ultra-rápida usada para armazenar dados frequentemente acessados.

Avanços Recentes

  • Multi-Core: Processadores com múltiplos núcleos (ex. Intel i9, AMD Ryzen 9).
  • Arquitetura de 7nm e 5nm: Transistores menores para maior eficiência (ex. Intel, Apple M1).

2. Memória

A memória de um computador armazena dados temporários ou permanentes necessários ao funcionamento do processador. Ela é crucial para o desempenho geral do sistema.

Como Funciona a Memória

A memória é dividida em duas categorias principais:

  • Memória Volátil (RAM): Perde os dados quando o computador é desligado. Tipos principais incluem DRAM e SRAM.
  • Memória Não Volátil: Mantém os dados após o desligamento, como ROM, Flash, HDDs e SSDs.

Componentes Eletrônicos da Memória

  • Células de Memória: Armazenam os dados usando transistores e capacitores (DRAM) ou flip-flops (SRAM).
  • Controlador de Memória: Gerencia a comunicação entre o processador e a memória.

Inovações Recentes

  • Memórias SSD: Utilizam chips Flash para maior velocidade e durabilidade.
  • DDR5: A nova geração de RAM, mais rápida e eficiente.
  • HBM (High Bandwidth Memory): Usada em placas gráficas e sistemas de alto desempenho.

3. Sistemas Operacionais

Um sistema operacional (SO) gerencia os recursos do computador e facilita a interação entre o usuário e o hardware.

Funções Principais

  • Gerenciamento de Processos: Controla a execução de programas.
  • Gerenciamento de Memória: Controla o uso da RAM.
  • Gerenciamento de Arquivos: Organiza dados no armazenamento.
  • Segurança e Controle de Acesso: Impede acessos não autorizados.

Exemplos de Sistemas Operacionais

  • Windows
  • Linux
  • macOS
  • Android e iOS

4. Sistemas Embarcados

Os sistemas embarcados são projetados para realizar funções específicas, como controlar um micro-ondas, um automóvel ou dispositivos IoT.

Características dos Sistemas Embarcados

  • Hardware Específico: Projetado para uma tarefa única.
  • Software Integrado: Software otimizado para tarefas específicas.
  • Recursos Limitados: Memória e processamento limitados.
  • Alta Confiabilidade: Necessário em dispositivos críticos.

5. Princípios de Segurança da Informação

A segurança da informação protege os dados contra acessos não autorizados e ataques.

Triade CIA

  • Confidencialidade: Garantir que os dados sejam acessados apenas por usuários autorizados.
  • Integridade: Garantir que os dados não sejam alterados sem autorização.
  • Disponibilidade: Garantir que os dados e sistemas estejam disponíveis quando necessário.

Outras Práticas de Segurança

  • Autenticação e Autorização: Garantir que apenas usuários e dispositivos verificados acessem os dados.
  • Criptografia: Protege os dados durante a transmissão ou armazenamento.
  • Firewalls e IDS: Monitoram tráfego e detectam atividades maliciosas.

Sistemas Embarcados e Segurança

Nos sistemas embarcados, a segurança pode ser um desafio devido a recursos limitados. Contudo, com a popularização da IoT, a segurança desses dispositivos se tornou uma preocupação crescente.
