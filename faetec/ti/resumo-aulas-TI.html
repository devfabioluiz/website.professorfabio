Resumo das aulas

TI

Operações Básicas

  • Inserir Dados: Digite diretamente nas células ou copie/cole de outras fontes.
  • Formatar Células: Acesse o menu Formatar para ajustar fontes, bordas e preenchimentos.
  • Classificação e Filtro: Ordene e filtre dados utilizando as opções no menu Dados.
  • Proteger Planilhas: Proteja células ou planilhas inteiras para evitar alterações acidentais.

Principais Funções

  • SOMA: =SOMA(A1:A10) - Soma os valores de um intervalo.
  • MÉDIA: =MÉDIA(A1:A10) - Calcula a média dos valores.
  • SE: =SE(A1>10;"Sim";"Não") - Avalia condições lógicas.
  • PROCV: =PROCV(10;A1:B10;2;FALSO) - Busca valores em uma tabela.
  • CONCATENAR: =CONCATENAR(A1;" ";B1) - Junta textos de células diferentes.
  • CONT.VALORES: =CONT.VALORES(A1:A10) - Conta células não vazias.
  • HOJE: =HOJE() - Retorna a data atual.
  • MAIOR: =MAIOR(A1:A10;1) - Retorna o maior valor de um intervalo.

Ferramentas Essenciais

Gráficos

Permite criar representações visuais de dados. Passos:

  • Selecione os dados desejados.
  • Acesse Inserir > Gráfico.
  • Escolha o tipo de gráfico (barra, linha, pizza, etc.).
  • Personalize cores e rótulos conforme necessário.

Tabelas Dinâmicas

Ferramenta para resumir e analisar grandes volumes de dados.

  • Acesse Dados > Tabela Dinâmica.
  • Defina o intervalo de dados e arraste os campos para criar relatórios personalizados.

Validação de Dados

Restringe entradas em células.

  • Selecione uma célula ou intervalo.
  • Acesse Dados > Validação.
  • Configure as regras, como números dentro de um intervalo.

Formatação Condicional

Permite destacar células com base em critérios específicos:

  • Selecione as células desejadas.
  • Acesse Formatar > Formatação Condicional.
  • Escolha o tipo de regra (maior que, menor que, entre, etc.).
  • Defina os valores e a aparência desejada (cor de fundo, fonte, etc.).

Dicas Finais

  • Use atalhos como Ctrl + C (copiar) e Ctrl + V (colar) para maior agilidade.
  • Salve arquivos regularmente para evitar perda de dados.
  • Explore extensões e plugins disponíveis para funcionalidades adicionais.
